# 🚗 AZAR RIF Cars App

Une plateforme complète de location de voitures.

## 🧩 Technologies
- Backend : Flask + MongoDB
- Frontend : React
- Authentification + Gestion des réservations + Upload de contrats

## ▶️ Démarrage local

```bash
cd backend
pip install -r requirements.txt
python app.py

cd ../frontend
npm install
npm start
